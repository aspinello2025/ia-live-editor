Como Funciona a Instalação e Uso?

Nenhuma configuração complexa é necessária. O plugin é instalado e ativado em menos de 1 minuto.

01

Instale o Plugin

Suba o arquivo ZIP na área de plugins do seu WordPress e faça a ativação.

02

Ative sua Chave

Insira a sua chave de ativação única no painel de licenças do plugin.

03

Abra no Frontend

Navegue pelo site normal. Você verá o botão "Editar conteúdo" na barra superior.

04

Salve Instantaneamente

Clique em qualquer bloco para editar e aperte "Salvar" para aplicar no Elementor.

Perguntas Frequentes

Tire suas dúvidas antes de garantir a sua licença de uso do plugin.

Preciso da versão paga do Elementor Pro?
Não! O plugin IA Live Editor funciona perfeitamente tanto com a versão gratuita do Elementor quanto com o Elementor Pro.
As edições salvam de verdade no Elementor?
Sim. Diferente de outras soluções, nosso plugin localiza a estrutura do Elementor salva no banco de dados daquela página específica, edita o campo do widget correspondente e atualiza no banco de dados. Quando você abrir o painel do Elementor tradicional, as edições estarão salvas lá exatamente da mesma forma.
Como funcionam as atualizações automáticas?
Assim que realizamos atualizações ou correções, uma notificação tradicional de nova versão aparece no painel principal do seu WordPress. Basta clicar em "Atualizar agora" para atualizar o plugin automaticamente do nosso repositório.
O plugin deixa o site lento para os visitantes?
Absolutamente não! O código do editor visual e todos os seus recursos adicionais (scripts, fontes e estilos) só são carregados quando você está logado como administrador do site e ativa explicitamente o modo de edição. Para o visitante comum da página, o plugin carrega zero scripts a mais.
